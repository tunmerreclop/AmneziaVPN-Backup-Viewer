# AmneziaVPN Backup Viewer

Просмотр файла бэкапа AmneziaVPN и извлечение информации, в частности, поиск информации касательно VLESS (amnezia-xray). Поддерживает подписку AmneziaVPN Premium с включенным VLESS.

> [!NOTE]
> 👇 Пояснение от автора 👇

Написали небольшой сайтик, извлекли сервера свои из подписки AmneziaVPN Premium. Можно сторонние и self-hosted сервера извлекать.

https://tunmerreclop.github.io/AmneziaVPN-Backup-Viewer/

Для AmneziaVPN Premium:
1. Включаем VLESS в подписке в приложении.
2. Выбираем страну.
3. Настройки → Резервное копирование → Создать резервную копию.

Теперь можно перегнать все сервера в VLESS. Просто идём сразу со второго пункта.

Если СТОРОННИХ/SELF-HOSTED добавленных серверов в AmneziaVPN много:
Хотите извлечь все сервера из премиума – рекомендую сначала сделать резервную копию, потом удалить все сервера, которые не входят в подписку AmneziaVPN Premium, после чего уже делать все махинации. После осуществления бэкапа последнего сервера из подписки – восстановитесь из первого бэкапа.

Для сторонних серверов/self-hosted можно просто сделать бэкап (Настройки → Резервное копирование → Создать резервную копию), ничего более делать не нужно.
